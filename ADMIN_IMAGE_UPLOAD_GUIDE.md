# Admin Panel Image Upload Guide

## 📸 Image Specifications for Each Section

### 1. **BANNERS** (Homepage Carousel)

#### Desktop Banner Image
- **Recommended Size**: 1920 x 1080 px (16:9 aspect ratio)
- **Alternative Sizes**: 1600 x 900 px or 1440 x 810 px
- **File Size**: Keep under 500 KB for fast loading
- **Format**: JPG or PNG (JPG recommended for photos)
- **What It Looks Like**: Full-width hero banner at the top of the homepage
- **Screen**: Displays on laptops, tablets (768px and above)

#### Mobile Banner Image (Separate)
- **Recommended Size**: 1080 x 1920 px (9:16 aspect ratio)
- **Or Portrait**: 768 x 1024 px
- **File Size**: Keep under 300 KB
- **Format**: JPG or PNG
- **What It Looks Like**: Tall vertical banner for mobile phones
- **Screen**: Displays only on mobile screens (below 768px)
- **Tip**: If not provided, desktop image will be used as fallback

**Pro Tip**: Use same spiritual image/theme for both desktop and mobile for consistency!

---

### 2. **GALLERY** (Image Gallery/Lightbox)

- **Recommended Size**: 1200 x 800 px (3:2 aspect ratio)
- **Alternative**: 1024 x 768 px or 1440 x 960 px
- **File Size**: Keep under 400 KB
- **Format**: JPG or PNG (JPG recommended)
- **What It Looks Like**: Images displayed in a grid that opens in lightbox when clicked
- **Screen**: Responsive - works on all devices
- **Aspect Ratio**: All images should have similar aspect ratios for clean grid appearance

**Good For**: Temple ceremony photos, deity darshan photos, event highlights, prasadam preparation

---

### 3. **EVENTS** (Event Cards)

- **Recommended Size**: 800 x 600 px (4:3 aspect ratio)
- **Or Square**: 800 x 800 px (1:1 aspect ratio)
- **Alternative**: 640 x 480 px
- **File Size**: Keep under 300 KB
- **Format**: JPG or PNG
- **What It Looks Like**: Displayed as event card thumbnail on /events page and homepage
- **Screen**: Responsive - adjusts for mobile and desktop
- **Crop Style**: Images will be cropped with object-cover, so place important content in center

**Good For**: Festival images, event posters, special occasion photos

---

### 4. **BLOG/ARTICLES** (Featured Images)

- **Recommended Size**: 1200 x 630 px (Facebook/Blog standard)
- **Alternative**: 1024 x 576 px or 1440 x 810 px
- **File Size**: Keep under 350 KB
- **Format**: JPG or PNG
- **What It Looks Like**: Featured image at top of blog posts and in blog listing
- **Screen**: Responsive on all devices

**Good For**: Spiritual quotes, teachings, festival information, temple updates

---

### 5. **QR CODE & PAYMENT IMAGES** (Bank Details)

- **Size**: Any size is fine (will be optimized automatically)
- **File Size**: Keep under 200 KB
- **Format**: PNG (best for QR codes), JPG, or WebP
- **Recommended Resolution**: 400 x 400 px to 800 x 800 px
- **What It Looks Like**: QR code for UPI payments, bank details image

---

## 🎨 General Best Practices

### File Size Optimization
| Image Type | Recommended Max Size | Format |
|---|---|---|
| Banners | 500 KB | JPG |
| Gallery | 400 KB | JPG |
| Events | 300 KB | JPG |
| Blog | 350 KB | JPG |
| QR Codes | 200 KB | PNG |

### Format Recommendations
- **JPG**: Use for photos, ceremonies, events (smaller file size)
- **PNG**: Use for logos, graphics, QR codes (supports transparency)
- **WebP**: Best quality, smallest size (but less browser support)

### Screen Resolutions to Consider
- **Mobile**: 375px wide (iPhone SE)
- **Tablet**: 768px wide (iPad)
- **Desktop**: 1920px wide (Full HD)
- **Large Desktop**: 2560px wide (4K screens)

**Mobile First**: All images are designed mobile-first, then enlarged for desktop

## ✅ Upload Checklist

Before uploading an image, make sure:
- [ ] Image is in correct format (JPG/PNG)
- [ ] File size is under recommended limit
- [ ] Image is clear and high quality
- [ ] Content is centered (important for event/gallery images)
- [ ] Image is properly oriented (not rotated)
- [ ] No blurry or low-resolution images
- [ ] Aspect ratio matches recommendations above
- [ ] Image reflects ISKCON spiritual themes

## 💡 Pro Tips

1. **Use online tools to compress images**:
   - TinyPNG.com - Great for JPG/PNG compression
   - Squoosh.app - Google's image optimizer
   - ImageOptim (Mac) or FileOptimizer (Windows)

2. **Resize images before uploading**:
   - Use Preview (Mac), Paint (Windows), or online tools like Pixlr.com
   - Resizing = smaller file size = faster website

3. **Batch process multiple images**:
   - If uploading gallery with 10+ images, resize all to same dimensions first
   - Creates consistent, professional-looking grid

4. **Mobile images should be portrait**:
   - Use vertical/portrait orientation for mobile banners
   - Use horizontal/landscape for desktop banners

5. **Test how images look**:
   - After uploading, view on mobile and desktop
   - Check if image is clear and not pixelated
   - Verify text/details are readable

---

## 🔍 If Images Look Blurry or Stretched

1. **Blurry on Desktop**: Upload higher resolution version (2400 x 1350 for banner)
2. **Stretched**: Check aspect ratio - use recommended dimensions exactly
3. **Cut Off**: Make sure important content is in center of image
4. **Colors Look Wrong**: Convert to sRGB color profile in image editor
